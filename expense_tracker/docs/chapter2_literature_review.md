# 2. Literature Review

## 2.1 Existing Systems

### 2.1.1 Commercial Solutions

1. **Mint**
   - Pros:
     - Comprehensive financial tracking
     - Bank integration
     - Budget recommendations
   - Cons:
     - Limited international support
     - Privacy concerns
     - Advertising-based model

2. **YNAB (You Need A Budget)**
   - Pros:
     - Strong budgeting methodology
     - Educational resources
     - Real-time sync
   - Cons:
     - Steep learning curve
     - Subscription-based
     - Limited investment tracking

3. **Personal Capital**
   - Pros:
     - Investment focus
     - Retirement planning
     - Wealth management
   - Cons:
     - High-wealth target audience
     - Limited basic budgeting
     - Complex for daily use

### 2.1.2 Open Source Alternatives

1. **GnuCash**
   - Pros:
     - Free and open source
     - Double-entry accounting
     - Comprehensive reporting
   - Cons:
     - Desktop-only
     - Dated interface
     - Complex for casual users

2. **HomeBank**
   - Pros:
     - Simple interface
     - Basic budgeting
     - Multi-platform
   - Cons:
     - Limited features
     - No mobile support
     - Basic reporting
